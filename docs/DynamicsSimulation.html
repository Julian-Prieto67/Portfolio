n Double Compound Pendulums on a Cart

Skills involved

MATLAB and the Matlab Symbolic Toolbox System Simulation Generalized variables

Description

I was inspired by the strange phenomenon that occurs when multiple metronomes are released while attached to a freely sliding rigid body. The metronomes may start out of sync, but eventually, the bodies all influence each other such that they will oscillate at the same frequency. I was inspired by this phenomenon to create a simulation that could analyze the solution for a double compound pendulum instead. Each pendulum could be represented by two compound rods of mass mij where i takes on all values through n pendulums and j takes on the value 1 or 2 to describe each link. These n double pendulums are rigidly attached to a communal rigid body which moves freely on two massless wheels.

This project was done in MATLAB and used the matlab symbolic toolbox to arrive at an equation of motion that described the whole system. I could then plug in various initial conditions that erratically changed the behavior of the animation. Overall, I learned that the double compound pendulum’s inherently chaotic dynamics carried over to this experiment and I didn’t see much synchronous motion. Regardless, this was a great lesson in generalizing dynamic simulations.
